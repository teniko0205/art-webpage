# 藝術學習資源中心 - 專案規格說明書

## 專案概述
這是一個純前端的藝術學習資源網站，設計用於展示和分享藝術相關的教學內容、影片和藝術史知識。專案使用純 HTML、CSS 和 JavaScript 開發，並透過 CDN 引入必要的套件，以便輕鬆部署到 GitHub Pages。

## 專案目標與進度追蹤

### 主要目標
1. 建立一個易於使用的藝術學習平台
2. 提供多元且系統化的藝術教學資源
3. 協助使用者選擇合適的繪圖工具
4. 建立藝術史知識庫
5. 打造互動性高的學習社群

## 代碼文件說明

### 核心文件
1. `index.html`: 首頁、主要導航和內容展示

   - 收藏系統邏輯
   - 動畫控制
   - 頁面交互
3. `styles.css`: 自定義樣式補充

### 頁面文件
- `pages/`: 子頁面目錄
  - `course-*.html`: 課程詳情頁面
  - `tools-*.html`: 工具推薦詳情頁面
  - `favorites.html`: 收藏管理頁面

## 收藏功能技術實現
1. 資料結構
```javascript
favorites = {
    'item-id': {
        type: 'course|video|history',
        title: '項目標題',
        timestamp: 1234567890
    }
}
```

2. 存儲方式
- 使用 localStorage 進行本地存儲
- 鍵值：'favorites'
- 格式：JSON 字符串

3. 主要功能
- 收藏/取消收藏
- 收藏狀態即時更新
- 視覺化反饋（星形圖標變色）
- 操作提示（提示訊息）

4. 待開發功能

## 視覺設計
- 主色調：紫色系 (purple-600)
- 輔助色：粉色 (用於漸變)
- 背景：淺灰 (gray-50)
- 卡片：白色背景配合陰影效果
- 動畫：使用 AOS.js 實現滾動動畫
- 收藏按鈕：
  - 未收藏：灰色 (gray-400)
  - 已收藏：黃色 (yellow-500)
  - 懸停效果：顏色過渡動畫

## 部署資訊
1. 目前專案設計為純靜態網站，可直接部署到 GitHub Pages
2. 所有外部資源使用 CDN 引入，無需額外的構建步驟
3. 部署步驟：
   - 建立 GitHub 儲存庫
   - 上傳專案檔案
   - 在儲存庫設定中啟用 GitHub Pages
   - 選擇 main 分支作為來源
